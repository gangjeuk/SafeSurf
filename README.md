## 소개

SafeSurf는 크롬 웹 익스텐션에 AI 어시스턴트를 통합하여 사용자가 웹을 탐색 중 실시간 금융사기 위험을 분석·경고하는 서비스입니다.  
멀티모달 위협, 최신 사기 방식, 신뢰도 평가 등 실질적 피해 예방을 목표로 삼고 있습니다.

---

## 주요 기능

- 텍스트, 영상 제목·자막·음성·댓글 등 실시간 멀티모달 위험 탐지
- 페이지 내 사기 패턴/키워드 하이라이트
- AI 기반 정보 신뢰도 산출 및 경고 배너 표시
- 주요 금융 사기 동향 자동 수집·요약
- 금융 지식 챗봇(질의응답)
- 유사/사칭 도메인 탐지 및 경고
- 사용자의 관심/행동 패턴 기반 맞춤 경고
- 탐지된 위험 키워드·패턴 조회
- 위험 탐색 이력 시각화(심화)
- 이메일/메신저 링크 위험 검증(심화)
- AI 기반 금융 용어 자동 설명(심화)

## AI 아키텍처

- **멀티 에이전트 시스템:** Langchain과 Langraph 기반 다중 에이전트(Planner, Searcher, Navigator)  
- **검색증강 생성(RAG):** 금융감독원 등 기관 데이터가 VectorDB에 구축되어 신뢰도 향상  
- **주요 기술:** Langchain, Langraph, LLM, VectorDB

---

## 데이터 및 저장소

- **활용 데이터:**  
  - 공식 금융 사기 사례, 뉴스 기사  
  - 영상 플랫폼 멀티모달 정보(제목, 자막, ASR, 댓글, 이미지)  
  - 금융 사기 키워드·패턴 사전 ("원금 보장" 등 주요 사기 문구)

- **로컬 프라이버시 보호:**  
  사용자의 브라우징 데이터는 WebAssembly 기반 PostgreSQL VectorDB에 로컬 저장되며 외부로 유출되지 않습니다.

## 기대 효과

- **금전적 효과:**  
  월간 100만 활성 사용자가 30% 행동 변화 시 연 528억 원 이상 피해 예방 예상
- **사회적 임팩트:**  
  3년간 800억 원 규모의 피해 감소, 불법 금융방 피해의 3~4% 감소 목표

## 설치 안내

1. 이 저장소를 클론합니다.  
   ```
   git clone https://github.com/your-org/safesurf
   ```
2. Node 버전(권장: `.nvmrc` 파일 참고) 및 패키지 매니저(pnpm)를 준비합니다.
   ```
   npm install -g pnpm
   ```
3. 의존성을 설치합니다.
   ```
   pnpm install
   ```
4. 환경별 명령어로 개발 서버 또는 빌드를 실행할 수 있습니다.
   - 개발:  
     ```
     pnpm dev
     ```
   - 배포:  
     ```
     pnpm build
     ```
